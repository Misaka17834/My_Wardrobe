## 📖 项目简介

**My_Wardrobe** 是一款衣物数据统计助手———帮助你记录每一件衣物的各项数据，方便你在浩如烟海的网购平台选购合身的衣物（尤其是个儿矮的男生和个儿高的的女生）！在未来，My Wardrobe还将支持多设备互联、接入AI一键生成今日热门穿搭、多语言混编实现更加现代化的交互界面、一键导出为excel离线保存……

## 🗂️ 模块说明

| 目录 | 说明 |
|:---|:---|
| `config/` | 项目配置信息，包括版本号、路径配置等 |
| `core/` | 核心业务逻辑，数据库增删改查操作 |
| `gui/` | 所有界面相关的代码，使用 tkinter/ttkbootstrap |
| `userdata/` | 用户数据存储，每个衣柜独立文件夹 |
| `preview/` | 项目预览截图，用于 README 展示 |

## ✨ 核心功能

### 🗄️ 多衣柜管理

- **多衣柜支持** - 为不同亲友创建独立衣柜
- **一键切换** - 快速在不同衣柜间切换，数据完全隔离
- **安全删除** - 智能保护当前使用的衣柜，防止误删

### 👕 衣物品类管理

- **自定义分类** - 按上衣、裤子、鞋子、配饰等自由分类
- **拖拽排序** - 鼠标长按即可拖拽调整类型顺序
- **智能归类** - "全部"和"未分组"自动汇总，永不遗漏

### 📝 衣物信息管理

- **丰富参数** - 名称、品牌、颜色、尺码、购买日期、价格...
- **自定义字段** - 根据需求添加任意参数（如：适用季节、洗涤方式）
- **照片上传** - 为每件衣物添加照片，所见即所得
- **快速编辑** - 右键菜单支持重命名、删除等快捷操作

### 🖼️ 照片查看器

- **独立窗口** - 点击衣物即可查看照片
- **图片操作** - 支持旋转、缩放、拖拽
- **智能跟随** - 窗口位置自动跟随主窗口

### ⚙️ 系统设置

- **关于页面** - 版本信息、项目地址一键跳转
- **衣柜管理** - 新建、切换、删除衣柜
- **更多功能** - URL路由、Excel导出（开发中...）

## 🚀 快速开始

### 环境要求

- Python 3.8 或更高版本
- Windows / macOS / Linux

### 安装依赖

```bash
pip install ttkbootstrap pillow
```

### 运行程序

```bash
python main.py
```

首次启动将引导你创建第一个衣柜！

## 💡 使用技巧

| 技巧 | 说明 |
|:---:|:---|
| 🖱️ **拖拽排序** | 在类型列表中长按 300ms 后拖拽，可调整顺序 |
| ↔️ **列头拖拽** | 在表头长按 200ms 后拖拽，可调整列顺序 |
| 🖱️ **右键菜单** | 右键点击类型或列头，快速重命名/删除 |
| ⌨️ **回车确认** | 所有弹窗支持 Enter 键快速确认 |
| 📷 **照片查看** | 点击衣物行即可在右侧查看照片 |
